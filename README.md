# OpenCV Color Recognition

## Project Description
This project detects colors in an image using OpenCV. It identifies the following colors:
- Red
- Green
- Blue
- Yellow

A green rectangle is drawn around each detected object.

## Requirements

- Python 3.x
- OpenCV
- NumPy

Install the required packages:

```bash
pip install -r requirements.txt
```

## Project Files

- main.py : Main Python program
- test.jpg : Test image
- requirements.txt : Required libraries
- README.md : Project documentation

## How to Run

1. Open the project folder.
2. Activate the Anaconda environment.
3. Run:

```bash
python main.py
```

The program will open a window showing the detected colors with green bounding boxes.
